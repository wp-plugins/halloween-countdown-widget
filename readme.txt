== Description ==

The Halloween Countdown Widget displays a cute vampire countdown to Halloween in your sidebar. 

 More information at [ChristmasWebmaster.com](http://christmaswebmaster.com/).

== Installation ==



1. Upload `hallocount.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your widgets screen, and drag the Halloween Countdown widget to your sidebar.
4. Let the countdown to Halloween begin!



== Frequently Asked Questions ==



= Does the bat have a name? =

Of course he does! We call him Vampy.

= Can I change the Countdown text color? =

Yes. You can edit the hallocount.php file CSS to suit your needs, find this line: color: #FFFF00; and edit accordingly.
